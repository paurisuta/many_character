<h1>概要</h1>

このサービス（メニキャラ）ではキャラを固定化されることなくチャットを楽しもうというサービスです。

<h1>何ができるのか（機能紹介）</h1>

メニキャラでは以下のことができます。

<ul>
  <li>ポストの投稿/削除</li>
  <li>ユーザーの登録/編集/削除</li>
  <li>ログイン機能</li>
  <li>フォロー機能</li>
  <li>タイムラインの表示</li>
  <li>お気に入り機能</li>
</ul>


<h1>開発環境<h1>

<ul>
  <li>AWS（cloud9）</li>
  <li>ruby 2.6.3</li>
  <li>rails　5.2.4.1</li>
  <li>heroku</li>
</ul>
